# 督友 APP・完整规划方案（存档版）

## 一、产品初心

从自身自律痛点出发，先解决自己拖延、难以坚持习惯的问题，再帮助同类人群；用最低成本、最自然的方式做一款真正有用的工具，不盲目追商业化，不一开始就收费。

## 二、产品定位

- 名称：督友
- Slogan：有人盯，才坚持；坚持着，就遇见
- 核心：**隐私习惯打卡 + 微信群强监督 + 轻度社交激励**
- 覆盖人群：需要自律的成人、学生、上班族，单身 / 非单身、家庭、熟人小圈都适用
- 端侧：iPhone APP 为主 + 微信小程序辅助提醒

## 三、核心创新点

1. **隐私优先**
    
    习惯内容、目标、进度仅自己可见，群内只同步「已打卡 / 未打卡」状态，不泄露任何隐私。
2. **微信群强监督**
    
    利用微信高频打开习惯，定时 @未打卡成员，用社交压力形成 “强迫症式自律”，不尴尬、不冒犯。
3. **双社交场景**
    
    - 熟人社交：家人、情侣、朋友、同事小群，互相监督，高粘性
    - 陌生人社交：按兴趣 / 地域 / 习惯匹配，轻交友，不搞颜值擦边
    
4. **奖惩驱动坚持**
    
    坚持打卡解锁社交权限、匹配机会；漏打卡冻结部分功能，利用损失厌恶倒逼行动。

## 四、核心功能（MVP + 完整版）

### 1. 最小可用版（先给自己用）

- 私密习惯创建与管理
- 一键每日打卡
- 连续天数 / 日历热力图
- 微信群状态同步（仅打卡与否）
- 定时群内 @提醒
- 简单统计与漏打卡提示

### 2. 完整版功能

- 多习惯分类：早起、健身、学习、视频创作、副业、作息等
- 熟人小群 / 家庭群 / 兴趣群 / 同城群
- 陌生人兴趣匹配与轻聊天
- 连续打卡保护、补签卡
- 自律等级、魅力值、成就勋章
- 群数据周报 / 月报
- 保证金挑战瓜分

## 五、商业模式（国内友好版：先免费，后温和变现）

### 阶段 1：0~3 个月 完全免费

- 所有核心打卡、建群、提醒、基础匹配全免费
- 目标：让自己先用顺，让用户上瘾、裂变，不设置任何付费门槛

### 阶段 2：3~6 个月 小额可选付费

- 主题皮肤、个性名片
- 超大群权限、数据导出
- 匹配加速卡等轻量增值服务

### 阶段 3：6 个月后 稳定变现

1. VIP 会员：解锁更多匹配、无限制聊天、优质群
2. 高级群 / 家庭群付费
3. 打卡保证金瓜分（平台抽成 10% 左右）
4. 高质量兴趣群入群费

### 核心逻辑

参考钉钉模式：**基础免费留人，高级功能付费增收，用社交 / 圈子压力锁留存**，不强制付费，不影响核心体验。

## 六、双社交场景设计

1. **熟人社交（底盘与留存）**
    
    家人健康监督、情侣互相督促、朋友学习 / 减肥搭子、同事小团队目标管理；粘性极高、几乎零运营、靠口碑裂变。
    
2. **陌生人社交（增长与收入）**
    
    按同习惯、同城匹配，以自律程度为推荐标准，而非颜值；坚持打卡才能解锁更多社交机会，把 “变好” 和 “交友” 绑定。
    

## 七、产品优势与壁垒

1. 市场空白：无同类「隐私打卡 + 微信群监督 + 自律交友」产品
2. 人性驱动：利用懒、好面子、渴望同频社交、怕损失等底层心理
3. 低风险：不擦边、不过度社交，微信 / 苹果审核友好
4. 可轻可重：自己能用，小群能用，大规模用户也能用

## 八、开发与上线思路

1. 先做极简 MVP，只满足自己日常自律使用
2. 接入微信群机器人做状态同步与提醒
3. 小范围给朋友试用，迭代优化
4. 稳定后再逐步加社交、付费功能
